# Prerequisites (Hardware)

Before installing Smyles Station, make sure you have the right hardware and understand the system requirements.

## Minimum System Requirements

### Computer Specifications

**Operating System:**
- Windows 10 or later (64-bit)
- Ubuntu 20.04 or later
- Other modern Linux distributions

**Processor:**
- Intel Core i3 or equivalent AMD processor
- 2.0 GHz or faster

**Memory (RAM):**
- 4 GB minimum
- 8 GB recommended (especially for multiple simultaneous games)

**Storage:**
- 500 MB for Smyles Station application
- Additional space for temporary files and cache
- SSD recommended for better performance

**Display:**
- 1024x768 minimum resolution
- 1920x1080 (Full HD) recommended
- Touchscreen optional but recommended for younger children

**Network:**
- Internet connection required for web-based games
- Wired Ethernet recommended for reliability
- WiFi acceptable with strong signal

## Recommended Hardware Setup

### Ideal Kiosk Computer

For the best experience, we recommend:

**Desktop Computer:**
- Modern desktop or all-in-one PC
- Intel Core i5 or AMD Ryzen 5 processor
- 8 GB RAM
- 256 GB SSD
- 24" touchscreen monitor (if separate)

**Why Desktop over Laptop?**
- More durable for public use
- Easier to secure physically
- Better cooling for all-day operation
- More ports for peripherals

### Peripherals

**Keyboard:**
- Wired keyboard recommended (wireless can be disabled by kids)
- Spill-resistant or washable design
- Consider a silicone cover for protection

**Mouse:**
- Wired optical mouse
- Cleanable surface
- Large buttons for small hands
- Consider a trackball for durability

**Touchscreen (Optional but Recommended):**
- More intuitive for young children
- Reduces need for mouse/keyboard
- Easier to clean than traditional input devices

## Network Requirements

### Internet Connection

**Bandwidth:**
- Minimum: 5 Mbps download
- Recommended: 10+ Mbps download
- Upload speed less critical

**Why you need good bandwidth:**
- Web-based games stream content
- Some educational sites use video
- Multiple users may share the connection

### Network Access

**Considerations:**

1. **Firewall:** Ensure your firewall allows access to whitelisted game sites
2. **Content Filter:** Content filters should allow educational game sites
3. **Network Isolation:** Consider separate guest network for kiosk
4. **Static IP (optional):** Makes remote management easier if needed

## Power Management

### Power Supply

**Requirements:**
- UPS (Uninterruptible Power Supply) recommended
- Protects against power outages during sessions
- Prevents data corruption from sudden shutdowns
- Allows graceful shutdown during extended outages

**Surge Protection:**
- Minimum: Surge protector power strip
- Better: UPS with surge protection
- Protects expensive equipment from power surges

### Power Saving

Smyles Station includes automatic shutdown scheduling, but consider:

- Set computer to never sleep during operating hours
- Enable sleep mode after-hours (or use scheduled shutdown)
- Disable monitor sleep during active sessions
- Configure wake-on-LAN if you want remote power management

## Budget Considerations

### New Hardware Costs (Approximate)

**Budget Setup ($400-600):**
- Refurbished desktop computer
- Basic monitor
- Keyboard and mouse
- Surge protector

**Mid-Range Setup ($800-1,200):**
- New desktop computer
- 24" touchscreen monitor
- Quality peripherals
- UPS backup

**Premium Setup ($1,500-2,500):**
- High-performance all-in-one PC
- Commercial-grade touchscreen
- Kiosk enclosure
- Professional installation

### Cost-Saving Options

1. **Refurbished Computers:**
   - Often available from reputable vendors
   - Meet minimum requirements
   - Significantly lower cost

2. **Donated Equipment:**
   - Many organizations receive computer donations
   - Verify meets minimum requirements
   - May need RAM/SSD upgrades

3. **Existing Hardware:**
   - Repurpose an existing public computer
   - Install Smyles Station alongside other software
   - Use kiosk mode to prevent access to other programs

## Accessibility Considerations

### Physical Accessibility

- Adjustable-height desk for wheelchair users
- Touchscreen at reachable height
- Large-button keyboard for motor difficulties
- Trackball mouse as alternative to standard mouse

### Software Accessibility

Smyles Station supports:
- Screen reader compatibility (via browser)
- High-contrast mode (via system settings)
- Keyboard navigation
- Adjustable text size (in web games)

## Common Hardware Questions

**Q: Can I use an old laptop?**
A: Yes, if it meets minimum requirements. Desktop is preferred for durability and security.

**Q: Do I need a touchscreen?**
A: Not required, but highly recommended for younger children.

**Q: What about Chromebooks or tablets?**
A: Smyles Station requires Windows or Linux. Chromebooks and tablets are not currently supported.

**Q: Can I use wireless keyboard/mouse?**
A: Not recommended - tech-savvy kids may disable them. Wired peripherals are more secure.

**Q: What if my computer doesn't have enough RAM?**
A: Many older computers can be upgraded. Check if your computer has available RAM slots and upgrade to at least 8 GB for best performance.
